== Description ==
Northern-Clouds is a simple html5 / CSS3 2column theme. It use an Ubuntu Font. A horizontal version of 'Superfish' jQuery drop-down menu is built in. It supports custom header and backgrounds. Translation Ready. Optimized view for Smartphones and Tablets.
